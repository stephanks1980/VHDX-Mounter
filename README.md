# VHDX-Mounter
Mount VHDX Images.
Created with AI.
After installation just right click on vhdx file and select open with VHDX Mounter.
All PArtition inside file will be mounted.
To unmount just do a right Click and secect Mount with VHDX Mounter again.
You will be asked to unmount. All Partitions in a vhdx will be unmounted.
Eject Button in File Manager does not work...
Tested on Debian 13 with Nemo.
